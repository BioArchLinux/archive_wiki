    贡献

    lilac

    Packages 仓库是由 lilac 构建的。 lilac 通过 lilac 相关文件，如 lilac.yaml 和 lilac.py，进行更新。它同时允许用户通过 PKGBUILD 构建仓库。

    PKGBUILD

    PKGBUILD 书写指南由 ArchWiki 的 PKGBUILD page 提供，你可以发现不同编程语言书写的软件有不同书写 PKGBUILD 的方式。

    当然，也可以参考 AUR ，如果你是一个新手。当然， Arch Forum 和我们的 discussions 是个讨论这些问题的好地方。

    lilac.yaml

    lilac.yaml 是 lilac 的必要文件。一般性的 书写指南 由 archlinuxcn 提供。API 被列在这里。Q&A 在这里可以被找到。

    对于 lilac.yaml 的书写，请参考 archlinuxcn 仓库 和 arch4edu 仓库。

    lilac.py

    lilac.py 是 lilac 的非必要文件。lilac.py 的书写指南由 archlinuxcn wiki 提供。

    对于 lilac.py 的书写，请参考 archlinuxcn 仓库 和 arch4edu 仓库。
